BACKUP

Qui puoi gestire il backup della tua collezione. Puoi creare un nuovo backup della collezione, dove verranno copiati i dati delle monete, le immagini e le fatture per poi essere compressi in un file zip. Puoi ripristinare i dati da un backup vecchio, selezionando il file zip contenente il backup. Dopo un ripristino, tutti i dati attuali vengono rimossi.
